# eDock-CryptoTool

eDock-CryptoTool is a simple encryption and decryption app for programmers.

## What it does

- Encrypt text and files
- Decrypt protected content
- Help with basic secure data handling in development workflows

## Why use it

Use it when you need a quick way to protect sensitive strings, notes, or project data without leaving your coding environment.

## Features

- Simple encrypt/decrypt workflow
- Built for programmer use
- Lightweight and easy to use

## Usage

1. Open eDock-CryptoTool.
2. Enter the content you want to encrypt or decrypt.
3. Choose the action.
4. Copy the result and use it in your project.

## Notes

This tool is designed for quick local use in developer workflows.
